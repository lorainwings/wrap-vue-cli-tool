A command line tool that encapsulates vue-server-cli help us build multiple pages project;

### Installation

<a name="installation"></a>

```shell
npm install wrap-vue-cli -D
```

```javascript
 wrap-vue-cli serve
 wrap-vue-cli build
 wrap-vue-cli start
```
